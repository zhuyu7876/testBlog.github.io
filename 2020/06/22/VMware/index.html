VMware 中 Mac OS 10.14 虚拟机安装教程【图文】（多图）

穷学生一个，一直没有体验过使用Mac是啥感觉，只好用虚拟机来过一把黑苹果的瘾。

网络教程或多或少都有错误，索性自己参考摸索总结。

安装环境

  1. Win10

  2. VMware Workstation Pro 15.0.0

    下载地址：http://download3.vmware.com/software/wkst/file/VMware-workstation-full-15.0.0-10134415.exe

    （许可号：AC11H-4HZ05-08EDQ-APQGX-YCUC8）

  3. Unlocker for VMware（补丁）

  4. macOS10.14 / macOS 10.13（任选其一）

    下载地址：https://pan.baidu.com/s/1UKDif7ZbsT42w-1pSDYjsA

    提取码：0mm9

虚拟机配置

  • VMware安装跳过，自行百度教程
  • 安装支持macOS的补丁Unlocker

  1. 打开任务管理器停止所有VMware相关的服务与进程

  1. 解压unlocker.zip，以管理员权限运行win-install.cmd，命令框自动消失则为安装成功
  • 安装 macOS 10.14

  1. 创建新的虚拟机

  2. 自定义

  3. 选择系统文件

  4. 选择安装路径

  5. 处理器、内存视电脑性能确定，可默认

  1. 网络配置（NAT）

  2. 下面全部选推荐即可

  3. 磁盘容量，视电脑情况

  4. 不要马上启动！还需修改配置文件！！！

    在如图位置添加 smc.version = “0”

  1. 到这里算是基本安装完成，启动虚拟机。

macOS系统配置

  1. 看到这个启动界面是不是有点激动~

  2. 接着一路默认“继续”即可

  1. 抹盘

    点击安装界面的下一步(右箭头图标)，选中【实用工具】中的【磁盘工具】继续，点击【显示所有卷宗】，选中要安装的硬盘后点击【抹掉】即可，可参考下图(不同版本的磁盘工具大同小异)。

  2. 安装

    抹盘完成后，点击【退出磁盘工具】，选择【安装macOS】继续，会让你选择要安装的分区(下图左)，选择刚刚抹掉硬盘分区即可进入安装过程(如下图右上)，安装完成后会提示设置地区、键盘、输入法等信息。

  1. 当当当~，现在已经正式安装好了

  2. 老是感觉分辨率有问题，看着不舒服

    关闭mac

    VMware Tool安装：进入设置，选择Unlocker/tools/payload下的darwin.iso

  3. 启动虚拟机，点击桌面VMware Tools进行安装

  4. 重启，记得这里设置分辨率

    到这里分辨率问题解决了，是16:9，可以全屏了，舒服。

  5. 现在开始配置网络

    windows系统win+R输入services.msc进入服务，开启DHCP Service和NAT Service，确认状态处于“正在运行”

  6. OK，可以上网了。

  • 安装结束
  • 可能是电脑配置不高，运行起来还是有点卡的，不过不计较啦，能白嫖体验macOS已经很香了，还要啥自行车。
